# 🧠 System Overview

**MANIHUB Healthcare Management Protocol** is a modern healthcare administration platform designed to streamline hospital operations, patient monitoring, appointment scheduling, and departmental management.

The platform delivers a responsive and intelligent healthcare ecosystem with modern dashboard interfaces and secure medical management protocols.

# 🛠️ Installation Protocol

## 1️⃣ Clone Repository

```bash
git clone https://github.com/22230902mani/HEALTHCARE_MANAGEMENT-.git
cd HEALTHCARE_MANAGEMENT-
```

---

## 2️⃣ Setup Backend

```bash
Install XAMPP / WAMP
Move project folder into htdocs
Start Apache & MySQL
```

---

## 3️⃣ Configure Database

- Open phpMyAdmin
- Create database
- Import SQL file
- Configure database connection

---

## 4️⃣ Launch Application

```txt
http://localhost/HEALTHCARE_MANAGEMENT-/
```
